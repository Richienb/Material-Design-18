Modals

Modals are very useful, when it comes to user notifications or quick input handling. This library includes many different modals, listed belove.

Add the class "modal-toggle" to the element you want to trigger the modal and fill the "modal-toggle" attribute with the id of the modal, you want to open. The modal div has to be outside of every other dividers.

Javascript

You can toggle the modal using JavaScript

You can also toggle if the modal is closeable. By default, its true.
