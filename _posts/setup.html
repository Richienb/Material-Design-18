Setup material design

Go through the steps of installing Material '18 library.

  1. To set up Material '18, you first have to download the library, from here. Keep in mind, we only support the latest release of Material '18!
  2. After you downloaded it, extract it to your project folder. The project structure will look like this:
  3. Now you have Material '18 installed. You can now use it 2 ways. You can use the default layout (used by the index.html file) or you can create a brand new file and include the css and js libraries.
